# Milestones

## M1: 기술 검증 (Week 1)

- GitHub Copilot SDK 인증/스트리밍 POC
- 계정 연동 및 entitlement 체크
- SDK 사용 가능 여부 `go/no-go` 판단

## M2: 기본 UX (Week 2)

- 사이드바 채팅 뷰
- 스트리밍 응답 렌더링
- 세션 생성/전환/삭제

## M3: 컨텍스트/변경 적용 (Week 3-4)

- 활성 노트/선택 텍스트 컨텍스트 주입
- 변경 제안 diff 프리뷰
- 적용/취소/되돌리기

## M4: CLI 기능 동등성 강화 (Week 5)

- 핵심 명령 플로우 매핑
- 설정 화면(모델/권한 정책)
- 오류 복구 플로우 정리

## M5: 안정화/베타 (Week 6)

- 성능 개선
- 로그/진단 정리
- 베타 배포 및 피드백 수집
